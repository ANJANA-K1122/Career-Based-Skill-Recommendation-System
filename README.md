# Career-Based Skill Recommendation System

An AI-powered web application that recommends skills and generates personalized career roadmaps using LSTM-based machine learning.

## Features
- Career skill recommendation
- Personalized roadmap generation
- Course recommendations
- Skill progress tracking
- Login and signup system

## Technologies Used
- Python
- Flask
- TensorFlow
- LSTM
- HTML
- CSS
- JavaScript
- SQLite

## Run Project

Install requirements:

pip install -r requirements.txt

Run Flask server:

python app.py
